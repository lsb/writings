We don't need to call each bookstore more than once, assuming the shop workers are all accurate, so we can split the task among as many people as we want.

Amazon Mechanical Turk

We can farm out our repetitive maintenance checks to the same round-the-clock labor pool that Amazon uses.

Strangers do work for pennies →magic→ quality!

So we need to get confidence in a heterogeneous labor pool performing work of uneven quality.

Standard approach:

Add a few questions
with known answers.

If you know the answers for a few questions in a batch of work, extrapolate the quality on those questions to the entire batch of work.
